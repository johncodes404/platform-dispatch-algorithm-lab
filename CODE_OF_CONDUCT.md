# 社区行为准则

我们希望这里成为一个尊重劳动经验、欢迎跨学科讨论、允许善意分歧的学习社区。

## 我们鼓励

- 使用包容、清晰和有依据的语言；
- 认真倾听劳动者、学习者和一线教师的经验；
- 对观点和实现提出具体意见，而不是评价个人；
- 主动说明假设、资料来源和不确定性；
- 接受不同年龄、专业、能力和技术背景的参与者；
- 在发现错误时帮助修复，并给新贡献者留下可理解的上下文。

## 不可接受

- 骚扰、羞辱、歧视、威胁或人身攻击；
- 公开他人的私人信息；
- 将群体经验简单标签化，或以教学之名贬低任何职业；
- 捏造数据、研究出处或真实平台内幕；
- 破坏项目、恶意灌水或持续打断协作；
- 对未成年人进行不适当接触或收集其个人信息。

## 处理方式

如果发现不当行为，请通过仓库维护者提供的私密联系方式报告；在私密渠道建立前，可在不披露个人信息的前提下创建 Issue，请求维护者联系。维护者可根据严重程度编辑或删除内容、关闭讨论、拒绝贡献或限制参与。

维护者应尽量说明处理依据，同时保护报告者和相关人员的隐私。

本准则适用于仓库、Issue、PR、讨论区以及代表本项目开展的线上线下活动。
